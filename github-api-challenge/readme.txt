This is a spring boot application.
To Start the Application,

1. Run MainApp.java with Run As -> Spring boot Application
2. Once the server started successfully, Open Postman Client
3. Hit the url in below format
		http://localhost:8080/{desired GIT id}/getFollowers
	Ex: http://localhost:8080/mediaczar/getFollowers

4. You will get the list of Git Followers
